# @daflan-org/falcon-compliance-toolkit

Falcon CI pipeline'inda policy input ve test evidence artefaktlarini standard sekilde
ureten komut satiri araci.

## Commands

- `falcon-compliance-toolkit build-policy-input`
- `falcon-compliance-toolkit run-test-evidence`
- `falcon-compliance-toolkit run-e2e-test-evidence`
- `falcon-compliance-toolkit merge-test-evidence-manifests`
- `falcon-compliance-toolkit validate-test-evidence`
